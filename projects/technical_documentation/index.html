Introduction

Python is a popular programming language created by Guido van Rossum.

It is widely used for web development, automation, and data analysis.

print("Hello, world!")
  • Simple syntax
  • Readable code
Syntax

Python syntax is clean and easy to learn.

Indentation is used to define code blocks.

if x > 5:\n print("x is large")
  • No semicolons
  • Uses indentation
Variables

Variables do not need declaration in Python.

They are created automatically when a value is assigned.

x = 5
  • Dynamically typed
  • Can change type
Functions

Functions are defined using the def keyword.

You can pass arguments and return values.

def greet(name):\n return "Hello " + name
  • Reusable code
  • Organized logic
Loops

Python supports for and while loops.

They are used to iterate over data or run logic multiple times.

for i in range(5):\n print(i)
  • for loop
  • while loop
